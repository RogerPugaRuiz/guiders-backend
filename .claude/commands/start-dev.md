Inicia el servidor de desarrollo de NestJS en segundo plano.

Ejecuta `npm run start:dev` en background para que puedas seguir trabajando mientras el servidor está corriendo.

Usa el comando Bash con `run_in_background: true` para ejecutar:
```bash
npm run start:dev
```

Después de iniciarlo, informa al usuario el ID del proceso para que pueda monitorear la salida con BashOutput o detenerlo con KillShell.
